# 💼 Molazmat App - QA Test Cases

## 📌 Project Overview
This project focuses on designing comprehensive test cases for the Molazmat job portal application. The aim was to validate the core functionality, usability, and reliability of the platform from a user perspective.

## 🧠 Application Type
Job Portal / Recruitment Platform

## 🧪 Testing Scope

- UI/UX Testing
- Form Validation
- Search & Filter Testing
- User Flow Testing

- Validated job search and application workflows
- Identified edge cases and input validation issues

- User registration & login
- Job search functionality
- Filters (location, category, salary)
- Job application process
- Form validation and error handling
